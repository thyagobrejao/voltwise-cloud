# VoltWise Cloud

Central backend and API for the VoltWise platform.

This service manages users, chargers, sessions, and integrations.

## ⚙️ Features (planned)

- User management
- Charger management
- Charging sessions
- Global charger map
- Public API
- Billing integration

## 🛠️ Tech Stack

- Django
- Django REST Framework
- PostgreSQL
- Redis
- Celery

## 🔗 Integrations

- voltwise-core
- voltwise-ocpp
- voltwise-agent

## 🚀 Goal

Serve as the central brain of the VoltWise ecosystem.
